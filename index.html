⚙️ Guia de Configuração do Firebase (único passo necessário)

  1. Acesse console.firebase.google.com e faça login com sua conta Google.
  2. Clique em "Criar um projeto", dê um nome (ex: quiz-sala) e conclua.
  3. No menu lateral, clique em "Realtime Database" → "Criar banco de dados".
    Escolha qualquer região → selecione "Iniciar no modo de teste" → Ativar.
  4. No menu lateral, vá em "Configurações do projeto" (ícone ⚙️) → aba "Geral".
    Em "Seus aplicativos", clique em "</>" (Web), registre um nome e copie o objeto firebaseConfig.
  5. Abra este arquivo HTML num editor de texto, localize o bloco const firebaseConfig = { e substitua cada "COLE_AQUI" pelo valor correspondente.
  6. Salve e abra o arquivo no navegador — a mensagem abaixo vai sumir e o jogo vai funcionar em qualquer dispositivo!
